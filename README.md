# milan-pm

> Milan Perl Mongers


## How to

Per creare il sito in locale ti serve Ruby,
<!-- stavo per mettere la foto di Ruby :) -->

e installare le dipendenze

```bash
gem install bundler
bundle install
```

Avvia il server e prova la generazione dei contenuti

```bash
jekyll serve --watch
```

Vedi [Using Jekyll with Pages][1] come riferimento.

  [1]: https://help.github.com/articles/using-jekyll-with-pages "Using Jekyll with Pages"
